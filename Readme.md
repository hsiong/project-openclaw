# File

## 🧠 一、核心人格 /身份层（最重要）

### 1️⃣ `SOUL.md`

👉 **人格 / 行为风格 / 输出方式**

**作用：**

- 决定“怎么说话”
- 语气、风格、思考方式
- 是否严谨 / 简洁 / 冷静 / 工程化

**例子：**

```
你是一个严谨的工程助手，优先给出结论，其次解释原因。
避免废话。
```

**特点：**

- ✅ 强影响输出风格
- ✅ 每次对话都会用
- ⭐⭐⭐⭐⭐（最重要）

------

### 2️⃣ `IDENTITY.md`

👉 **“你是谁”**

**作用：**

- 定义 agent 身份
- 名字、角色、职责

**例子：**

```
你是 OpenClaw 内部运维助手，负责处理系统配置问题。
```

**区别于 SOUL：**

- IDENTITY = **身份**
- SOUL = **性格 +行为方式**

------

### 3️⃣ `USER.md`

👉 **用户画像 / 偏好**

**作用：**

- 让模型“记住用户习惯”
- 输出语言、风格、技术水平

**例子：**

```
用户是后端工程师，偏好中文回答，熟悉 Docker 和 Java。
```

**效果：**

- 自动用你喜欢的语言（比如你现在）
- 少解释基础概念

------

## 🧠 二、记忆层（长期状态）

### 4️⃣ `memory/`（目录）

👉 **长期记忆系统（按日期）**

你日志里也出现了：

```
/workspace/memory/2026-03-18.md
```

**作用：**

- 每天生成一个 memory 文件
- 存摘要 / 关键事实

**例子：**

```
用户今天在配置 Ollama + OpenClaw
遇到 contextWindow 问题
```

**特点：**

- 自动读最近几天
- 防止上下文爆炸
- ⭐⭐⭐⭐

------

### 5️⃣ `HEARTBEAT.md`

👉 **运行时状态 / 心跳信息（弱记忆）**

**作用：**

- 存当前 session 的状态
- 有点像“临时脑子”

**常见内容：**

```
当前任务：调试 openclaw 配置
状态：进行中
```

**特点：**

- 比 memory 短期
- 更偏“当前任务”

------

## 🧠 三、系统控制层（很关键但容易忽略）

### 6️⃣ `AGENTS.md`

👉 **Agent 行为规则（系统级）**

这是很多人忽略但很重要的文件。

**作用：**

- 控制 agent 能做什么
- 工具使用策略
- 执行逻辑

**例子：**

```
优先直接回答问题
只有必要时才调用工具
不要反复调用同一个工具
```

**影响：**

- 是否乱用工具
- 是否过度推理
- 是否啰嗦

⭐ 实际上这是“调教 agent”的核心之一

------

### 7️⃣ `TOOLS.md`

👉 **工具使用说明（tool calling prompt）**

**作用：**

- 告诉模型怎么用 tools
- 什么场景用哪个工具

**例子：**

```
当需要读文件时，使用 read 工具
不要猜测文件内容
```

**影响：**

- tool 调用是否准确
- 是否 hallucination

------

### 8️⃣ `BOOTSTRAP.md`

👉 **启动时注入的系统 prompt（全局初始化）**

**作用：**

- 每次 session 开始时加载
- 类似“system prompt”

**例子：**

```
始终优先给出结论
避免冗长解释
```

**和 SOUL 的区别：**

| 文件      | 作用      |
| --------- | --------- |
| BOOTSTRAP | 全局规则  |
| SOUL      | 性格/风格 |

------

### 🧠 总结一张表（最清晰）

| 文件         | 作用        | 优先级 | 是否关键 |
| ------------ | ----------- | ------ | -------- |
| SOUL.md      | 人格/风格   | ⭐⭐⭐⭐⭐  | 必备     |
| IDENTITY.md  | 身份        | ⭐⭐⭐⭐   | 推荐     |
| USER.md      | 用户偏好    | ⭐⭐⭐⭐   | 推荐     |
| memory/      | 长期记忆    | ⭐⭐⭐⭐   | 推荐     |
| HEARTBEAT.md | 当前状态    | ⭐⭐⭐    | 可选     |
| AGENTS.md    | 行为策略    | ⭐⭐⭐⭐⭐  | 非常关键 |
| TOOLS.md     | 工具规则    | ⭐⭐⭐⭐   | 重要     |
| BOOTSTRAP.md | 全局 prompt | ⭐⭐⭐⭐⭐  | 核心     |
